# 🧪 Swift Interview Questions — Examples & Playground

A collection of Swift and SwiftUI code examples covering common interview topics — written as clean, annotated Playgrounds and Swift files you can study, run, and adapt.

---

## 🤔 What this is

This repo is a practical study guide for Swift and SwiftUI interviews. It covers the concepts that come up most in iOS developer interviews — value vs reference types, optionals, protocols, closures, concurrency, and SwiftUI patterns — with short, focused code examples for each. Everything is written to be read and understood quickly, not just copied.

## ✅ Why you'd use it

- **Interview prep that actually runs** — examples are in Playgrounds so you can tweak and experiment directly
- **Covers the essentials** — value types, reference types, protocols, generics, closures, `async/await`, and SwiftUI state
- **Annotated code** — comments explain *why* each pattern works, not just what it does
- **Quick reference** — scan a topic before an interview or use it to fill gaps in your Swift knowledge

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/SwiftInterviewQuestionsExamples.git
cd SwiftInterviewQuestionsExamples
```
Or select "Clone Git Repository…" when Xcode launches.

### 2. Open in Xcode
- Double-click the `.xcodeproj` or `.xcworkspace`.

### 3. Browse the Files
- Most examples are Swift Playgrounds — open them and hit **Run** to see output
- No signing or device required for Playground files
